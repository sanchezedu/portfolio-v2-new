# Portafolio de Sanchezedu

Este proyecto es mi portafolio personal como desarrollador web. Aqui muestro mi trabajo, mis servicios y la forma en que ayudo a emprendimientos a transformar ideas en productos digitales reales.

## De que trata

- Presenta mis proyectos con enfoque en resultados y experiencia de usuario.
- Explica mis servicios (landing pages, dashboards, sitios corporativos y soluciones con base de datos).
- Incluye planes claros con beneficios y enfoque comercial.
- Integra formulario de contacto para cotizaciones.

## Stack principal

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Nodemailer

## Como ejecutar el proyecto

1. Instala dependencias

```bash
pnpm install
```

2. Levanta el servidor de desarrollo

```bash
pnpm dev
```

El sitio estara en http://localhost:3000

## Variables de entorno

Para el formulario de contacto necesitas configurar las variables de Nodemailer en un archivo `.env`.

- `NODEMAILER_USER`
- `NODEMAILER_PASS`
